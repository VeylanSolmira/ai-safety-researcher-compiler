Topic Details

1. Agent Architectures & Design

Focus: Understanding how autonomous AI agents are structured

  • ReAct, Chain-of-Thought, Tool-augmented LLMs
  • Build a simple agent with safety constraints
  • Design patterns that promote safe behavior

2. Agent Safety Fundamentals

Focus: Core safety challenges unique to agents

  • Goal specification, reward hacking, instrumental goals
  • Implement safety monitors and kill switches
  • Preventing unintended autonomous actions

3. Agent Evaluation & Testing

Focus: Assessing agent behavior and capabilities

  • Sandboxed environments, behavioral unit tests
  • Create test suites for agent safety properties
  • Detecting deceptive or harmful behaviors

4. Tool Use & External APIs

Focus: Managing agent interactions with external systems

  • API security, rate limiting, action authorization
  • Build secure tool-use frameworks
  • Preventing unauthorized system access
